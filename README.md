# 🏸 羽毛球私人教练 App (Badminton Trainer)

一个专为**业余中级及以上球友**打造的顶级羽毛球辅助训练应用。这不仅是一个打卡工具，而是一个集成了**沉浸式跟练、智能节拍器、2.5D 高级鹰眼演示、多球随机报点、以及运动伤病预警**的完整教练系统。

**核心理念：** 一切数据留存本地，完全离线可用，无网络延迟。只要你把它带到地下二层的球馆，点下“开始”，剩下的全部交给虚拟教练。

---

## 🌟 核心特色亮点

### 1. 🎧 沉浸式虚拟教练带练 (Keep 同款)
- **5 秒就位倒数**：每组动作开始前，伴随清脆的“5、4、3...”语音倒数，给你足够时间走入场地。
- **自动分组合计**：针对“波比跳 4x15个”、“六点步法 5x30秒”这种动作，教练会精确帮你数数、倒计组内时间，并在组与组之间**自动提供 30 秒休息倒数**，休息结束自动唤起下一组！
- **智能随机报点**：在练步法时，虚拟教练会在背景音中冷不防地随机喊出：“左前网”、“右接杀”、“左后退”... 完美模拟场边真实教练给你喂球的条件反射训练。
- **动态状态缩放**：今天实在太累？训练前点一下“疲惫 (50%)”，系统自动将原定 60 分钟的计划智能压缩为 30 分钟核心训练。

### 2. 🎬 2.5D 高级教学动画与手绘演示
- **鹰眼 3D 轨迹系统**：完全抛弃模糊的网路视频，使用 React Native 矢量引擎纯手工渲染 2.5D 立体球场。你能看到不同球路在空中的真实抛物线、高度投影、和击球人挥拍的动态瞬间！
- **真实多球落点散布**：高远球/杀球等动画每次飞行的落点，会在“正手底角”和“反手底角”之间发生符合实战逻辑的随机偏移。
- **八边形握拍透视**：每个教程配备特制的球拍截面图，高亮显示拇指/食指的精确发力位置。
- **体能拟人动画**：体能训练配备了极其直观的小人动作（波比跳、深蹲、跳绳、平板等），以及不同强度的“呼吸光环”。

### 3. 🎵 视听动感双重刺激
- **离线内置 BGM**：包含动感电子、轻松纯音、史诗激昂三种风格的无版权高质音乐，可在跟练过程中无缝下拉切换，拒绝枯燥。
- **真实打击音效**：挥拍时伴随木质击球声“啪！”，步法移动时伴随球鞋摩擦木地板的“吱——”声。
- **全局触觉震动 (Haptics)**：每一次计步、跳过、打卡成功，都配有不同频率的物理震动，打击感十足。

### 4. 📊 职业级防伤病数据统计
- **A:C Ratio 运动伤预警**：采用体能界的“急慢性负荷比”，对比你近一周与近一月的训练量。若负荷超标突增，图表变红，强力警示你小心跟腱或肌肉拉伤。
- **GitHub 热力图**：你的 90 天训练打卡被可视化为“小绿块日历”，逼迫你填满它。
- **战绩复盘**：实战打卡不仅记时长，还可以标注你的对手和输赢战绩。

## 🛠 安装与使用说明

此项目已完美支持直接生成 Android 原生 APK 进行安装体验，所有视觉、音效、震动等高级原生特性均可完美运行。

### 方式一：下载云端 APK (最快体验)
通过 EAS (Expo Application Services) 的云端编译，你可以直接获得无需任何配置的纯净 Android 安装包。
1. 在项目目录确保代码已更新。
2. 运行打包命令：
   ```bash
   npx eas-cli build -p android --profile preview
   ```
3. 根据给出的控制台链接，在网页上下载 `.apk` 并安装到手机即可。

### 方式二：本地局域网真机调试 / Web 预览
若不想安装 APK，只要电脑和手机在同一 WiFi 下即可直接开发调试：

```bash
# 安装依赖
npm install

# 启动本地服务 (带原生功能需使用 Expo Go)
npx expo start --lan

# 若只在电脑浏览器中进行纯 Web 预览：
npx expo start --web
```
> **注：** Web 预览模式下，由于浏览器安全限制，部分原生特性（触觉震动、相机捕获等）会自动降级或被安全隔离。要体验完美的手感请务必使用真机 App。

## 🧩 技术架构

- **框架**: Expo SDK 52 + React Native 0.76 (新架构)
- **UI & 路由**: Expo Router (文件系统路由)
- **数据库**: `expo-sqlite` (本地长久存储，Web环境自动降级为 MemoryDB 保证跨平台通用)
- **动画引擎**: `react-native-reanimated` + `react-native-svg`
- **音频控制**: `expo-av` + `expo-speech` (系统级离线语音合成)
- **视觉底层 (维护中)**: `react-native-vision-camera` + `react-native-fast-tflite` (为未来版本接入真实 MediaPipe 动作分析留出预编译口)

本应用的数据（包括所有的训练打卡记录、你写下的笔记与复盘录像）**仅保存在您手机的 SQLite 数据库中**，绝不会上传至任何云端服务器，保障 100% 隐私安全。
